Introduction to HTML

HTML (HyperText Markup Language) is the standard language used to create web pages. It structures content on the web and consists of various elements that tell the browser how to display content

An HTML element is defined by a start tag(<), content, and an end tag(>).

DOCTYPE Declaration The <!DOCTYPE html> declaration defines the document type and version of HTML. It ensures the browser renders the page correctly.

The Head Element <head> element contains meta-information about the HTML document, such as the title, character set, and links to external resources (like stylesheets and scripts).

The Body Element <body> element contains the content of the HTML document. This includes text, images, links, tables, and other elements that are displayed on the webpage.

The Script Element <script> element is used to embed or reference executable scripts. This is typically used for client-side JavaScript to add interactivity to the webpage.

            
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Simple HTML Document</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <h1>Welcome to My Application</h1>
    <p>This is a paragraph of text.</p>
    <img src="image.jpg" alt="Description of image">
    <script src="script.js"></script>
</body>
</html>

Designing the layout with Bootstrap

Bootstrap is a popular framework for building responsive and attractive web layouts quickly. It provides a set of CSS classes and JavaScript components that can be easily integrated into your HTML.

1: Setting Up Bootstrap

To include Bootstrap in your HTML file, you should add the following <link> tag in the <head$gt; section of your HTML document:

            
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap Layout Tutorial</title>
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.0.2/dist/css/bootstrap.min.css">
</head>
<body>
